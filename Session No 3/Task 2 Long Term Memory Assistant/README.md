# Task 2 - Long Term Memory Assistant

## Objective

Create an AI assistant that stores user preferences in a JSON file and uses them to generate personalized responses.

## Features

- Save Preferences
- Load Previous Memories
- Personalized Responses
- Interactive Mode
- Gemini AI

## Technologies

- Python
- Gemini API
- JSON
- python-dotenv

## Run

```bash
python -m pip install -r requirements.txt
python main.py
```
