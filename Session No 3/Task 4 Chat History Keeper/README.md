# Task 4 - Chat History Keeper

## Objective

Build a chatbot that remembers previous user messages during a conversation and answers based on chat history.

---

## Features

- Stores conversation history
- Remembers user information
- Uses Gemini API
- Maintains memory during the session
- Provides a final memory summary

---

## Technologies Used

- Python
- Google Gemini API
- python-dotenv

---

## Project Structure

```
Task 4 Chat History Keeper

├── main.py

```

## Installation

