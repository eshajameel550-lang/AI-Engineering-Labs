# Session 2 – Prompt Engineering & AI Agents

##  Overview

In this session, I practiced different prompting techniques and learned how AI Agents perform reasoning, decision-making, and tool usage to solve problems.

---

##  Topics Covered

### 1. Chain of Thought (CoT) Prompting
- Learned step-by-step reasoning.
- Compared direct answers with reasoning-based responses.
- Solved logical and mathematical problems using CoT.

### 2. Few-Shot Prompting
- Learned how examples improve AI responses.
- Provided sample question-answer pairs.
- Generated answers in a consistent style.

### 3. ReAct (Reason + Act) Agent
- Understood the ReAct workflow.
- Implemented Thought → Action → Observation cycle.
- Used multiple tools like:
  - Search
  - Weather
  - Calculator
  - Translator
  - Time

### 4. AI Agent Practice
- Built simple AI Agent examples.
- Practiced decision making.
- Learned how agents choose tools based on user goals.
