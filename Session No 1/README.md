# Session No 1: Agentic AI Fundamentals

## Overview

This session introduces the core concepts of Agentic AI and Prompt Engineering. The practical tasks focus on building simple AI agents using the Google Gemini API, understanding how AI behavior can be controlled, and learning how agents interact with users through prompts, tools, and conversations.

---

## Learning Objectives

By completing this session, I learned how to:

- Connect Python with the Google Gemini API.
- Use environment variables to securely manage API keys.
- Design AI agents with different personas.
- Improve AI-generated responses using self-refinement.
- Integrate external tools with AI agents.
- Build interactive AI agents that ask questions before giving answers.

---

## Tasks Completed

### Task 1: Persona Expansion Class

**Objective:**
Create an AI agent that responds with different personalities by changing its role, tone, style, and constraints.

**Key Concepts:**

- AI Personas
- Prompt Engineering
- Dynamic Prompts
- Object-Oriented Programming (OOP)

---

### Task 2: Self-Refinement Loop

**Objective:**
Build an AI workflow where the model generates an initial response and then improves its own output.

**Key Concepts:**

- Self-Reflection
- Iterative Prompting
- Response Improvement
- AI Self-Critique

---

### Task 3: Agentic Tool Integration

**Objective:**
Enable an AI agent to use external tools for solving specific tasks such as mathematical calculations.

**Key Concepts:**

- Tool Calling
- Function Calling
- AI Agents
- Chat Sessions

---

### Task 4: Flipped Interaction

**Objective:**
Create a Travel Planner Agent that collects information step by step before generating a personalized travel plan.

**Key Concepts:**

- Multi-turn Conversations
- System Instructions
- Context Retention
- Interactive AI Agents
